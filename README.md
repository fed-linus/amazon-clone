# **Amazon Clone**

A simplified front-end clone of Amazon built using HTML, CSS, and JavaScript. This project demonstrates a responsive e-commerce layout with a header, search bar, product grid, and cart functionality.

---

## **Features**

- Responsive design that works on desktops, tablets, and mobile devices.
- **Header** with logo, search bar, cart, and orders section.
- **Product grid** dynamically populated using JavaScript.
- **Cart system** showing the quantity of items added.
- Styled using Google Fonts (Roboto) for a clean modern look.
- Fully modular structure with separate CSS and JS files.

---

## **Technologies Used**

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **LocalStorage** (for cart functionality)
- **Google Fonts: Roboto**
